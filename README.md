# The Artisan Kiln — Ceramic Tile Order Form

Фронтенд тестового задания: интерактивная форма заказа керамической плитки.

## Стек

- Next.js
- TypeScript
- Tailwind CSS
- Redux Toolkit
- Vitest для unit-тестов расчётов

## Что реализовано

- Адаптивная страница по макетам `design/design_desktop.png` и `design/design_mobile.png`.
- Корзина с изменением количества, удалением позиций и добавлением новой плитки.
- Динамические расчёты `Subtotal`, `Shipping`, `Grand Total`.
- Desktop-only дизайн-инструмент: сетка 7x7 и выбор плитки из палитры.
- Checkout form с переключением методов оплаты и клиентской валидацией.
- Глобальное состояние корзины, формы и визуализатора в Redux.
- Компонентная структура интерфейса с русскими комментариями в коде.

## Запуск

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Проверки

```bash
npm run build
```
